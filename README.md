# codigo-psico
## Resumen
Este es el codigo del programa del departamento de psicopedagogía de la UTD.

## Introducción
En el departamento de psicopedagogía de la universidad anteriormente mencionada existía un problema donde los registros se enredaban entre sí y no permitían el uso de las cuestiones generales del departamento. Este proyecto fue creado específicamente para ese propósito

## Aviso
En caso del uso de este archivo para otro propósito, debe de seguir la guía de la LICENCIA que se encuentra en el propio archivo para su distribución y modificación.
